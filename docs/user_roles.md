# User Roles & Permissions — PharmaQ

## Employee
- Start/attend vivas.
- Select question counts.
- Access study materials (locked during viva).
- View personal scores and weekly progress.
- Save/resume exams.
- Change password.

## Mentor
- Conduct viva sessions.
- Create/edit/delete questions.
- Manage study materials.
- View all employee scores.
- Set exam difficulty.
- Change password.

## Admin
- Full system control.
- Add/edit/delete Employees and Mentors.
- Role management (Mentor ↔ Employee).
- Configure difficulty logic.
- Manage question bank.
- Monitor weekly compliance and performance data.
